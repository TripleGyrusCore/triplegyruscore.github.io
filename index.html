Triple Gyrus Core

Triple Gyrus Core is a data and software system built from the ground up with accessibility, usability, and universal design in mind. Information is hard to access for many reasons, from file formats to assumptions about the abilities of people accessing the information. Triple Gyrus Core is built to address those concerns.

Version 1 of Triple Gyrus Core is focused on pulling data out of documents in an easy and accessible way. You can use Triple Gyrus Core to pull information out of images, complex documents, and PDFs using a markup overlay that is optimized for OCR reliablity and ease of use. The parts of the overlay are chosen to work well with left to right and right to left text, as well as assistive technologies.

If you are interested in using Triple Gyrus Core, it is free to use and always will be. If you would like implementation assistance in your workflows or would like to contribute to making Triple Gyrus Core better, please email triplegyruscore@gmail.com.
